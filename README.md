# README

Use script to download videos from youtube using yt-dlp, cut chunks based on time intervals provided in inputlist.txt, and combine into a single video

yt_dlp works better on residential IPs not datacentre IPs, but make sure your residential ISP can enough bandwidth + data cap to download lots of videos 

tested on macOS Sequoia 15.5, runs bash 3.2.57 (old version), on 2025-07-13
